# Workplace DBT PWA

鬼詰め・強い指摘の直後に、DBTのSTOP/TIPP、3分ログ、上司への返答文生成を使うためのスマホ向けPWAです。

## ファイル構成

- index.html: アプリ本体
- manifest.webmanifest: スマホのホーム画面追加・PWA表示用
- sw.js: オフライン利用用のService Worker
- icons/: ホーム画面用アイコン

## 使い方

1. このフォルダをGitHub Pages / Netlify / Vercelなどにアップロードします。
2. スマホで公開URLを開きます。
3. iPhoneの場合はSafariから「ホーム画面に追加」します。
4. Androidの場合はChromeから「アプリをインストール」または「ホーム画面に追加」します。

## 注意

- ログはスマホ内のブラウザ localStorage に保存されます。
- 機種変更、ブラウザデータ削除、別ブラウザ利用では引き継がれません。
- 機微情報を扱うため、社用PC・共有端末では使わないことを推奨します。
- 医療・診断・治療の代替ではありません。強い不調が続く場合は主治医・産業医・社内相談窓口へ接続してください。
