# 反芻ストッパー

負の反芻思考に入り始めたときに、スマホからすぐ使えるPWA形式のミニアプリです。

## 主な機能

- 「今は反芻モード」とラベルを貼る
- 冷水・早歩き・固定行動・呼吸など、身体から切り替える
- 60秒の呼吸リセット
- 事実／解釈／次の一手の3行メモ
- 心配を後で考える時間に予約
- ローカル履歴保存
- テキスト出力
- SOS時の簡易行動プラン

## スマホアプリ化する方法

### GitHub Pagesに置く場合

1. GitHubで新しいリポジトリを作成
2. このフォルダ内のファイルをすべてアップロード
3. Settings → Pages
4. Branchを `main`、Folderを `/root` にして保存
5. 公開URLをスマホで開く
6. Android Chromeなら「ホーム画面に追加」
7. iPhone Safariなら共有ボタン →「ホーム画面に追加」
