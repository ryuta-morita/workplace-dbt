状態チェックアプリ
==================

これは、スマホで使うことを想定した簡易PWA形式のセルフチェックアプリです。
医療診断ツールではありません。

判定仕様
--------
通常項目：9項目、各0〜3点、最大27点

0〜6点   ：安全
7〜13点  ：注意
14点以上 ：危険

以下のいずれかに該当する場合は、総点に関係なく「危険」：
- 希死念慮・自傷衝動が1以上
- 認知と事実の乖離・修整不可が3
- 被害妄想が3
- 息苦しさ、目眩、手の震え、頭痛のいずれかが3
- 焦燥感が3、かつ自己否定が2以上

使い方
------
1. index.html をスマホまたはPCのブラウザで開く
2. 各項目を0〜3点で選択
3. 「判定する」を押す
4. 必要なら「この結果を保存」を押す

Androidでアプリ風に使う方法
---------------------------
GitHub PagesなどにアップロードしてHTTPSで公開すると、Chromeのメニューから
「ホーム画面に追加」または「アプリをインストール」が使えます。

プライバシー
------------
履歴はブラウザのlocalStorageに保存されます。
外部送信はしていません。
